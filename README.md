## Project Description
This project recreates the front page of the historical newspaper *Hamara-Hindustan* using HTML and inline CSS. The layout was designed to resemble a traditional 1930s newspaper with structured columns, bold headlines, and boxed sections while following assignment constraints.

## Features Implemented
- Table-based newspaper layout for precise column alignment  
- Masthead with title and logo placement  
- Bold main headline and subheadline hierarchy  
- Boxed side article section  
- Two-column lower content structure  
- Shared section heading with separators  
- Inline CSS styling for fonts, borders, and spacing  S

## Challenges Faced
Recreating a historical newspaper layout using only HTML and inline CSS was challenging, especially maintaining column alignment. This was solved by using table-based structures and careful spacing adjustments.
